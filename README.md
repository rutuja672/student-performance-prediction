🤖 Student Performance Prediction

📌 Overview

This project uses Machine Learning to predict student exam scores based on study hours. A Linear Regression model is trained on historical data and used to estimate future performance.

🚀 Features

- Data preprocessing
- Linear Regression model
- Score prediction
- Data visualization
- Performance analysis

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

📊 Machine Learning Algorithm

- Linear Regression
